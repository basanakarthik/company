# Companies Directory (Frontend Developer Task)
This is a React + Vite + TailwindCSS frontend project for displaying and filtering companies.

## Features
- Responsive UI with TailwindCSS
- Search, filter, and sort companies
- Pagination
- Loading & error handling
- Mock API using static JSON

## Run Locally
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Build & Deploy
npm run build
Deploy `dist` folder on Vercel or Netlify.
